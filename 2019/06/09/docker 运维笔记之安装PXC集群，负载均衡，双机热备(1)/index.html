容器生命周期管理

命令 作用
docker run <镜像ID> 启动一个镜像到容器
docker start <容器ID> 启动指定容器
docker stop <容器ID> 停止指定容器
docker restart <容器ID> 重启指定容器
docker kill
docker rm <容器ID> 删除指定容器
docker pauser <容器ID> 暂停指定容器，配套unpausel
docker unpausel <容器ID> 恢复指定容器，配套pauser

容器操作运维

命令 作用
docker ps 查看正在运行的容器，docker ps -a 是查看全部容器
docker inspect <容器ID> 查看指定容器的配置等
docker top <容器ID> 查看容器中运行的进程信息
docker attach <容器ID> 从服务器获取实时事件
docker logs 获取容器的日志
docker wait 阻塞运行直到容器停止，然后打印出它的退出代码
docker export 将文件系统作为一个tar归档文件导出到STDOUT
docker port 列出指定的容器的端口映射，或者查找将PRIVATE_PORT NAT到面向公众的端口
docker rename 改名

容器rootfs命令

命令 作用
docker commit 从容器创建一个新的镜像。
docker cp 用于容器与主机之间的数据拷贝。
docker diff 检查容器里文件结构的更改。

镜像仓库

命令 作用
docker login 登录
docker pull 下载
docker push 提交
docker search 搜索

本地镜像管理

命令 作用
docker images 查看本地镜像
dcoker rmi 删除镜像
docker tag 改标签
docker build 创建一个镜像
docker history 查看镜像的创建记录
docker save 将指定镜像保存成 tar 归档文件。
docker import 从归档文件中创建镜像。

网络管理

命令 作用
docker network ls 查看网络信息
docker network create –subnet=网段 网络名称 ….
docker network rm 网络名称 删除某个网段

其他命令

命令 作用
docker version 查看版本
docker info 查看详细docker信息
